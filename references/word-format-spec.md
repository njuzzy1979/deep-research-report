# 研究报告 Word 版格式规范 v1.0

> ⚠️ **已废弃** — 本规范（v1.0）已被 [研究报告格式规范.md](研究报告格式规范.md)（V3.0，遨天科技编制，2026-06-24）取代。
> 请以 V3.0 规范为唯一格式依据。本文件保留仅供历史参考。
>
> **主要变更摘要**：
> - 字体体系：标题 黑体+Arial → 微软雅黑+Times New Roman
> - 字号：H1 16→24pt, H2 14→16pt, H3 12→14pt, 正文 12→11pt
> - 页边距：左 3.18→3.17cm, 右 3.18→2.54cm
> - 表格：从 Table Grid 默认样式 → 全框线含竖线、交替行灰底、自定义粗细
> - 封面：全新设计（标题 28pt Bold、报告类型标签、机构/版本信息）
> - 题注：表注 12pt 黑体加粗→9pt #555555，图注 10.5pt 楷体→9pt #555555
> - 页眉：居中→右对齐 + 1pt 黑色底线
> - 配色：全新配色方案（黑色 #000000 / #555555 / #F2F2F2 / #333333）

本规范适用于深度研究报告的 Word 文档输出，覆盖标题体系、正文、图表、引用等全部元素的字体/字号/行距/缩进/编号/大纲级别要求。

| 参数 | 值 |
|---|---|
| 纸张 | A4（210mm × 297mm） |

| 左/右边距 | 3.18cm（1.25 英寸） |
| 页眉距边界 | 1.5cm |
| 页脚距边界 | 1.75cm |
| 装订线 | 0cm |

## 二、字体方案

全文使用 4 种字体，不得超用：

| 角色 | 中文 | 英文/数字 | 说明 |
|---|---|---|---|
| **正文** | 宋体 | Times New Roman | 默认字体 |
| **一级标题** | 黑体 | Arial（加粗） | |
| **二/三级标题** | 黑体 | Arial（加粗） | |
| **摘要/引文/题注** | 楷体 | Times New Roman（斜体） | 特殊段落 |
| **代码/数据/网址** | 宋体 | Consolas | 等宽段落 |
| **强调** | 同上下文字体，**加粗**或*斜体* | 同上下文字体 | 不引入新字体 |

## 三、标题体系（大纲级别 + 字号 + 间距）

全文最多使用四级标题（含题名）。

### 3.1 报告题名（仅 1 个）

| 属性 | 值 |
|---|---|
| 字体 | 黑体 二号（22pt），加粗 |
| 对齐 | 居中 |
| 大纲级别 | 级别 1（但编号为"无"） |
| 段前 | 24pt |
| 段后 | 18pt |
| 行距 | 单倍行距 |
| 特殊格式 | 无 |
| 编号 | 无 |
| 示例 | `智能体敏捷适变模式分析及启示` |
| 副题名 | 黑体 小二（18pt），不加粗，居中，段后 12pt。格式：`——<副题名>`（破折号开头） |

### 3.2 一级标题（章标题）

| 属性 | 值 |
|---|---|
| 字体 | 黑体 三号（16pt），加粗 |
| 对齐 | 居中 |
| 大纲级别 | 级别 2 |
| 段前 | 18pt |
| 段后 | 12pt |
| 行距 | 单倍行距 |
| 编号格式 | `第X章`（X 为中文数字：一、二、三…） |
| 示例 | `第一章 2026年美以伊冲突背景分析` |

### 3.3 二级标题（节标题）

| 属性 | 值 |
|---|---|
| 字体 | 黑体 四号（14pt），加粗 |
| 对齐 | 两端对齐 |
| 大纲级别 | 级别 3 |
| 段前 | 12pt |
| 段后 | 6pt |
| 行距 | 1.25 倍行距 |
| 编号格式 | `X.Y`（阿拉伯数字，章号-节号） |
| 示例 | `2.1 高烈度非对称博弈` |

### 3.4 三级标题（小节标题）

| 属性 | 值 |
|---|---|
| 字体 | 黑体 小四号（12pt），加粗 |
| 对齐 | 两端对齐 |
| 大纲级别 | 级别 4 |
| 段前 | 6pt |
| 段后 | 3pt |
| 行距 | 1.25 倍行距 |
| 编号格式 | `X.Y.Z`（阿拉伯数字） |
| 示例 | `2.1.3 伊朗无人机对美军ISR链路的影响` |

### 3.5 禁止事项

- **禁止四级及以上标题**（不使用 4.1.1.1 类编号）
- 如果内容确实需要更细粒度，使用加粗段落首句而非编号标题
- 全文标题编号连续，不跳号、不重号
- 标题不以标点结尾（问号/感叹号例外）

## 四、正文段落

### 4.1 默认正文

| 属性 | 值 |
|---|---|
| 字体 | 宋体（中文）+ Times New Roman（英文/数字），小四号（12pt） |
| 对齐 | 两端对齐 |
| 大纲级别 | 正文文本 |
| 首行缩进 | 2 字符（即 24pt，12pt 字号 × 2） |
| 段前 | 0pt |
| 段后 | 0pt |
| 行距 | 1.5 倍行距 |

### 4.2 列表（项目符号/编号列表）

| 属性 | 值 |
|---|---|
| 字体 | 同正文 |
| 缩进 | 左侧缩进 1.5cm，悬挂缩进 0.75cm |
| 行距 | 1.5 倍行距 |
| 编号列表 | `1.` `2.` 或 `(1)` `(2)` 或 `①` `②` |
| 项目符号 | 实心圆点 `●`（首层）/ 短横 `–`（次层） |
| 层数限制 | 最多 2 层嵌套 |

### 4.3 特殊段落类型

| 类型 | 字体 | 缩进 | 用途 |
|---|---|---|---|
| **摘要段落** | 楷体 五号（10.5pt），首行缩进 2 字符 | 左右各缩进 1cm | 每章首段或全报告摘要 |
| **本章结论** | 楷体 小四号（12pt），首行不缩进，左侧设竖线边框 | 左侧缩进 1cm | 章开头的结论块 |
| **引文块** | 楷体 五号（10.5pt），首行缩进 2 字符 | 左右各缩进 1.5cm | 超过 3 行的直接引用 |
| **脚注** | 宋体 小五号（9pt） | 无缩进 | 页面底部注释 |
| **代码块** | Consolas 五号（10.5pt），加浅灰底纹 | 左缩进 1cm | 代码/命令/数据示例 |

## 五、图表体系

### 5.1 图表编号

格式：`图 X-Y` 或 `表 X-Y`（X = 章号，Y = 该章内序号，用阿拉伯数字）

- 全文统一按章编号，不从 1 开始跨章计数
- 图编号和表编号独立计数
- 附录图表用 `附图 A-1` / `附表 A-1` 格式

### 5.2 图题（图片下方）

| 属性 | 值 |
|---|---|
| 位置 | 图片正下方，居中 |
| 字体 | 楷体 五号（10.5pt） |
| 格式 | `图 X-Y <图名>` |
| 段前 | 6pt |
| 段后 | 12pt |
| 行距 | 单倍行距 |
| 示例 | `图 5-1 Palantir本体+AI核心架构图` |

> **图片本身**：居中，宽度不超过页面宽度的 90%，分辨率 ≥ 150dpi。SVG/PNG 优于 JPG。

### 5.3 表题（表格上方）

| 属性 | 值 |
|---|---|
| 位置 | 表格正上方，居中 |
| 字体 | 楷体 五号（10.5pt），加粗 |
| 格式 | `表 X-Y <表名>` |
| 段前 | 12pt |
| 段后 | 6pt |
| 行距 | 单倍行距 |
| 示例 | `表 6-2 TiDB与CockroachDB运维复杂度对比` |

### 5.4 表头行

| 属性 | 值 |
|---|---|
| 字体 | 黑体 小四号（12pt），加粗 |
| 对齐 | 居中（水平），居中（垂直） |
| 底纹 | 浅灰色（RGB: 240, 240, 240） |
| 行高 | 最小值 0.8cm |

### 5.5 表体行

| 属性 | 值 |
|---|---|
| 字体 | 宋体 小四号（12pt），英文/数字用 Times New Roman |
| 对齐 | 文本左对齐，数字右对齐，垂直居中 |
| 行高 | 最小值 0.6cm |
| 行距 | 单倍行距 |
| 边框 | 上下边框 1pt 实线，内部横线 0.5pt 虚线 |

### 5.6 图表来源注（表下方，如有）

| 属性 | 值 |
|---|---|
| 位置 | 表格下方，左对齐 |
| 字体 | 楷体 小五号（9pt） |
| 格式 | `数据来源：<来源描述>` 或 `注：<说明>` |
| 行距 | 单倍行距 |

## 六、引用与参考文献

### 6.1 正文内引用

| 属性 | 值 |
|---|---|
| 格式 | `[作者/机构，年份]` 或 `[媒体，日期]` |
| 字体 | 同正文 |
| 示例 | `据Reuters 2026年3月报道[Reuters, 2026]，Maven系统……` |

### 6.2 脚注式引用（可选，与正文内引用二选一）

| 属性 | 编排 | 值 |
|---|---|---|
| 编号 | 每页重新编号 | 阿拉伯数字，上标 |
| 字体 | 宋体 小五号（9pt） |
| 行距 | 单倍行距 |

### 6.3 参考文献列表（附录）

| 属性 | 值 |
|---|---|
| 标题 | 黑体 三号（16pt），居中，大纲级别 = 级别 2 |
| 条目字体 | 宋体 五号（10.5pt），英文/数字 Times New Roman |
| 编号 | `[1]` `[2]` `[3]`… 连续编号 |
| 悬挂缩进 | 1cm（编号悬于正文之外） |
| 行距 | 单倍行距 |
| 分类标题 | 黑体 小四号（12pt），段前 6pt。如"一、官方文件""二、媒体报道""三、学术论文" |

**条目格式**（参考 GB/T 7714-2015）：

- **期刊论文**：`[编号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.`
- **专著**：`[编号] 作者. 书名[M]. 出版地: 出版社, 年.`
- **新闻报道**：`[编号] 作者/媒体. 标题[N/OL]. 媒体名, 发布日期. URL.`
- **官方文件**：`[编号] 发布机构. 文件名[Z]. 发布日期.`
- **电子资源**：`[编号] 作者/机构. 标题[EB/OL]. 发布日期. URL.`

## 七、页眉与页脚

### 7.1 页眉

| 属性 | 值 |
|---|---|
| 奇数页 | 报告题名（楷体 小五号，居中） |
| 偶数页 | 当前章标题（楷体 小五号，居中） |
| 分隔线 | 0.5pt 实线，黑色 |
| 首页 | 不显示页眉 |

### 7.2 页脚（页码）

| 属性 | 值 |
|---|---|
| 位置 | 页脚居中 |
| 字体 | Times New Roman 小五号（9pt） |
| 摘要/目录 | 罗马数字：i, ii, iii… |
| 正文/附录 | 阿拉伯数字：1, 2, 3… |
| 首页 | 不显示页码（但计入计数） |

## 八、目录

| 属性 | 值 |
|---|---|
| 标题 | 黑体 三号（16pt），居中。文字：`目 录`（两字之间有空格） |

| 一级标题条目 | 黑体 小四号（12pt） |
| 二级标题条目 | 宋体 小四号（12pt） |
| 三级标题条目 | 宋体 小四号（12pt） |

| 页码 | 右对齐，前导符为 `……`（点式） |

## 九、分隔规则

| 场景 | 方式 | 说明 |
|---|---|---|
| 每章开始 | 分页符（新页开始） | 每章从奇数页开始（如条件允许） |
| 摘要与正文之间 | 分页符 | 摘要独立成页 |
| 正文与参考文献之间 | 分页符 | |
| 同一节内段落 | 不分页 | |
| 图表与所属段落 | 不分页 | 图表与首次引用它的段落必须在同一页或对开页 |
| 大节之间 | 空行（段后 12pt） | |
| 同级标题之间 | 空行（段后 6pt） | |

## 十、文件命名与导出

| 属性 | 值 |
|---|---|
| 文件格式 | `.docx`（Office Open XML） |
| 命名规则 | `<报告简称>_v<版本号>.docx`，如 `东南亚新能源汽车产业链研究_v1.0.docx` |
| 内嵌字体 | 不嵌入（用系统标准字体） |
| 兼容性 | 最低 Microsoft Word 2016 / WPS Office 2019 |
